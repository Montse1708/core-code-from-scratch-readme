<h2>3. ASCII Total exercise</h2>

`Solution`
```javascript
function uniTotal (string) {
// total up dem unicodes!
  let total = 0;
  for (let i = 0, length = string.length; i < length; i++) {
    total += string[i].charCodeAt();
  }
  return total;
}
```
