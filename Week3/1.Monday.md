<h2>2. Remove All Exclamation Marks From The End Of Sentence exercise</h2>

`Solution`

```javascript
function remove (string) {
  let result;
  let exclamationMark = string.length - 1;
  for (let i = exclamationMark; i > 0; i--){
    if (string[i] !== '!') {
      result = string.substring(0, i + 1);
      break;
    }
  }
  return result;
}
```
